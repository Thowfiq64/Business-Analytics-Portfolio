### Project Overview
This project analyzes a large transactional dataset from Amazon Fresh to identify revenue trends, customer purchasing behaviors, and inventory turnover rates. The goal was to solve critical business problems regarding stock management and sales optimization.

### Key Insights & Features
* **Sales Trends:** Identified peak sales periods and high-demand product categories.
* **Customer Behavior:** Analyzed purchase frequency and average basket size.
* **Inventory Optimization:** Flagged slow-moving inventory to reduce holding costs.

### Technical Skills Used
* **SQL:** Complex Joins, Aggregations, Subqueries, Window Functions.
* **Data Cleaning:** Handling NULL values and formatting date/time stamps.
* **Visualization:** Insights exported to Excel for reporting.
